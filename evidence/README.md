# Evidence records

This directory is the conventional drop point for evidence records emitted by your AI systems, when those records do not have a richer durable store (Postgres, S3, vendor SaaS).

For v0.1, the shape of an evidence record is fixed by `aigov.schema.Evidence`. Sites usually:

1. Emit records at runtime from the agent, gateway, or governance forum.
2. Send them to a durable store (Postgres is the default).
3. Use this directory only as a local drop for the starter pack generated by `aigov init`.

A typical record:

```python
from aigov import Evidence

ev = Evidence.now(
    id="audit-2026-06-14-000123",
    control_ids=["iso42001:A.6.2.7", "iso42001:A.6.2.5"],
    type="audit_log_reference",
    payload={"run_id": "demo-001", "audit_hash": "9f1a..."},
    source_system="agentic-harness",
)
```

ADR 0001 in `docs/adr/` explains why evidence cross-references the audit log rather than duplicating its payload.
